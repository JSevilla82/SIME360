En esta pestaña puede configurar las licencias de Microsoft 365 que el sistema usará para mostrar la información en otros módulos.

Tipos de licencia:
• Principal: Licencias esenciales para el entorno Microsoft 365 (Outlook, OneDrive, Teams, etc.).
• Secundaria: Licencias específicas para productos como Power BI Pro u otros servicios específicos.
